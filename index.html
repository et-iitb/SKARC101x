3D Architectural Visualization

Preface

Architectural visualization is a tool which helps to bridge the gap in understanding between architects, builders, and clients. Therefore, developing this skill is crucial if you are an architect in the making, or even an enthusiast. If the design is not communicated clearly through visualization, there can be a major impact.

This e-book covers the basic concepts of architectural visualization. We will learn about the stepwise processes to be followed to create a photo-realistic 3D visualization for the architectural drawing created by an architect/interior designer.

How to use this book

All the topics can be accessed from the Index page, which is accessible from the link in the lower section. To navigate to the next or previous pages, use the Next or Back arrows. Within each chapter are different Learning Dialogues (LeDs) or recorded lectures. Following each dialogue are practice questions. You can attempt the questions multiple times until you are clear with the concept.
